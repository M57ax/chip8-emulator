# Chip-8

Simple Chip 8 Emulator in C++ and SFML


## How does my function work
CPU = Chip8::cycle()
- gets Opcode
- execute Opcode 
- modified m_display

GPU(SFML) = drawDisplay() 
- reads m_display
- draws SFML Rectangle
